# PROJECT OVERVIEW - LCMS

## Dự Án: Hệ Thống Quản Lý Trung Tâm Ngoại Ngữ Thảo Nguyên

### Mục Tiêu
Xây dựng một hệ thống quản lý toàn diện cho trung tâm ngoại ngữ, kết nối:
- Website công khai
- Hệ thống đăng ký & thanh toán
- Learning Hub (nền tảng học tập)
- Quản lý giáo viên
- Quản lý điều hành (Manager)
- Quản trị hệ thống (Admin)
- Báo cáo tài chính

### Stack Công Nghệ
- **Frontend**: Next.js 15 + React + TypeScript + Tailwind CSS
- **Backend**: Supabase (PostgreSQL + Auth + Storage)
- **Deployment**: Vercel

### Phân Quyền
1. **Học Viên**: Learning Hub
2. **Giáo Viên**: Lớp phụ trách
3. **Quản Lý**: Toàn điều hành
4. **Quản Trị**: Hệ thống

### Giai Đoạn Phát Triển
1. Phân tích & thiết kế (Database, API)
2. Setup foundation
3. Core features
4. Learning Hub
5. Teacher Portal
6. Manager Portal
7. Admin Portal
8. Testing & Deployment
